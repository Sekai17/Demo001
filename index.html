¿Cómo está construida esta demo?

Esta página no es solo una demo visual, sino también una demostración de código limpio y buenas prácticas. Aquí están algunas de las técnicas clave utilizadas:

Tema Claro/Oscuro con CSS

El cambio de tema se logra con Variables CSS (ej: --bg-dark). Un simple script de JavaScript añade la clase .light-theme al <body>, lo que actualiza instantáneamente todos los colores definidos en las variables.

Contenido Dinámico con JS

Las tarjetas de "Proyectos" no están escritas en el HTML. Se crean dinámicamente usando JavaScript a partir de un array de datos. Esto hace que el contenido sea fácil de actualizar sin tocar la estructura HTML.

Formulario 100% Frontend

El formulario de contacto funciona sin un servidor. Guarda los datos en el localStorage de tu navegador y usa mailto: para abrir tu cliente de correo. Es una solución perfecta para prototipos o sitios estáticos.

Proyectos de Demostración

Estas tarjetas son generadas con JavaScript. Haz clic en cualquiera de ellas para abrir un modal interactivo que carga la información correspondiente del proyecto. Es una técnica muy común en aplicaciones web modernas.

Formulario de Contacto Funcional (Demo)

Este formulario demuestra cómo capturar y gestionar datos sin un backend. Al enviar, guardará una copia localmente (que puedes exportar como JSON) y preparará un borrador en tu aplicación de correo electrónico.
