¿Cómo se construyó esta demo?

Se aplicaron varias técnicas y buenas prácticas para asegurar que la página sea eficiente, moderna y fácil de mantener.

📄

Estructura en un Archivo

Para simplificar la portabilidad, todo el código (HTML, CSS y JavaScript) está en este mismo archivo. Esto es ideal para demos, prototipos rápidos y pruebas de concepto.

🎨

CSS Moderno y Theming

Se usan variables CSS (:root) para manejar los temas claro/oscuro de forma sencilla. El efecto "cristal" (glassmorphism) del header se logra con backdrop-filter para un toque visual moderno.

⚡️

JavaScript Vainilla

Toda la interactividad (modal, formulario, cambio de tema) se maneja con JavaScript puro, sin librerías externas. Esto optimiza el rendimiento y reduce el tiempo de carga de la página.

♿️

Accesibilidad y SEO

Se utiliza HTML semántico (<header>, <main>, <section>) y atributos como title y meta description para mejorar la accesibilidad para lectores de pantalla y la optimización para motores de búsqueda.

Proyectos Destacados (Demo)

Haz clic en cualquier proyecto para ver los detalles. La ventana modal que aparece es una demostración de manipulación del DOM con JavaScript para crear interfaces dinámicas sin recargar la página.

Formulario de Contacto "Serverless"

Este formulario es una simulación funcional. No requiere un servidor (backend). Al enviarlo, utiliza el protocolo mailto: para abrir tu cliente de correo. Además, guarda una copia de los mensajes en el localStorage de tu navegador, que puedes exportar como un archivo JSON.
